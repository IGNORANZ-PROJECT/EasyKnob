大切な注意事項

EasyKnobを使う前に、プライバシーと音量に関する注意を確認してください。

  • マイク音声はサーバーへ送信しません。音声処理はブラウザ内で行われます。
  • マイク権限が必要です。ONを押すと、ブラウザからマイク使用許可を求められます。
  • 音量に注意してください。ヘッドホンやスピーカーの音量を低めにしてから起動してください。
  • 外部アプリ側の設定も影響します。Discord、OBS、VRChat、VB-CABLE、BlackHoleなどの設定によって、音質や遅延が変わります。
  • 利用は自己責任です。聴覚、機器、配信、通話、OSやブラウザ設定に関する問題について、作者およびIGNORANZ PROJECTは責任を負いません。

この注意事項は初回のみ表示されます。詳しい使い方はHELPとREADMEを確認してください。

まず必要なもの

EasyKnob本体はブラウザで動きます。ただしDiscordやVRChatに渡すには、仮想オーディオが必要です。

  • Windows: VB-CABLE
  • Mac: BlackHole
  • 推奨ブラウザ: Chrome / Edge

Windows / VB-CABLE

  1. VB-CABLEをインストールします。
  2. EasyKnobのOutputを CABLE Input にします。
  3. Discordなどの入力を CABLE Output にします。

Mac / BlackHole

  1. BlackHoleをインストールします。
  2. EasyKnobのOutputを BlackHole にします。
  3. Discordなどの入力を BlackHole にします。

低遅延・音質

音質を優先する場合は、Discordや配信ソフト側のノイズ抑制、エコー除去、自動ゲイン調整をOFFにしてください。Windows側の音声拡張、空間オーディオ、メーカー製マイク補正もOFF推奨です。

遅延が気になる場合は、Windowsの「このデバイスを聴く」、Discordのマイクテスト、OBSの音声モニタリングなど、別経路の監視音をOFFにしてください。

EasyKnob側では、不要なSTABLE、ECHO、REVERB、ROOM、WET、AIR、DOUBLEをOFFまたは低めにすると原音に近くなります。AUDIO ANALYZERも不要な時はOFF推奨です。

各Knob(ノブ)の役割は、設定画面のノブ一覧に記載しています。

ハウリング対策

スピーカーから出した音をマイクが拾うとハウリングします。基本はヘッドホンか仮想オーディオ出力を使い、スピーカー監視は避けてください。HOWLが出る時は、MIC、WET、ECHO、REVERB、DOUBLEを下げ、Outputをスピーカー以外にしてください。

対応外環境について

Safariや一部ブラウザでは、出力先デバイスの選択が使えない場合があります。対応外環境では、EasyKnobを仮想マイク用途に使えない可能性があります。
