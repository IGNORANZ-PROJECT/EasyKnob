大切な注意事項

EasyKnobを使う前に、プライバシーと音量に関する注意を確認してください。

  • マイク音声はサーバーへ送信しません。音声処理はブラウザ内で行われます。
  • マイク権限が必要です。初回はONを押すまで許可画面は表示されません。ONを押した後、ブラウザのマイク使用を許可してください。
  • 音量に注意してください。ヘッドホンやスピーカーの音量を低めにしてから起動してください。
  • 外部アプリ側の設定も影響します。Discord、OBS、VRChat、VB-CABLE、BlackHoleなどの設定によって、音質や遅延が変わります。
  • 利用は自己責任です。聴覚、機器、配信、通話、OSやブラウザ設定に関する問題について、作者およびIGNORANZ PROJECTは責任を負いません。

この注意事項は初回のみ表示されます。詳しい使い方は「初めての方へ」とREADMEを確認してください。

ワンクリックで簡単設定

触らずに音を作る

用途に近いボタンを押すだけで、ノブ、ON/OFF、REVERB詳細をまとめて反映します。反映後に必要なところだけ微調整してください。

設定

ノブを無効化すると、メイン画面から隠し、音声処理から外します。

REVERB 詳細

REVERB成分だけにかかるEQです。原音にはかかりません。

初めての方へ

まず最初に

初回のマイク許可は、ONを押した後に表示されます。

Input、Output、Presetを選んでONを押してください。初回だけブラウザがマイク使用を確認します。許可すると状態がSTARTINGからLIVEに変わり、EasyKnobが動き始めます。

1 Chrome / Edgeで開く Output選択を使うため、デスクトップ版ChromeまたはEdgeがおすすめです。
2 InputとOutputを選ぶ Inputは使うマイク、OutputはCABLE InputやBlackHoleなどを選びます。
3 ONを押して許可 初回はここでマイク許可が出ます。許可しないと音声処理は開始されません。

Windows / Mac の接続

Discord、VRChat、OBSなどへ加工後の声を渡すには、仮想オーディオデバイスを使います。

Windowsの方
  1. ChromeまたはEdgeでEasyKnobを開きます。Safariや一部ブラウザではOutput選択が使えません。
  2. VB-CABLEの公式サイトからWindows用のZIPをダウンロードします。
  3. ZIPを右クリックして展開します。展開せずにZIP内から実行すると失敗する場合があります。
  4. 展開したフォルダ内の VBCABLE_Setup_x64.exe、環境によっては VBCABLE_Setup.exe を右クリックし、管理者として実行します。
  5. Install Driverを押し、完了したらWindowsを再起動します。
  6. EasyKnobのOutputを CABLE Input にします。
  7. Discord、VRChat、OBSなどの入力デバイスを CABLE Output にします。
  8. EasyKnobのONを押します。初回はここでブラウザのマイク許可が出るので、許可してください。
  9. 外部アプリ側のノイズ抑制、エコー除去、自動ゲイン調整はOFF推奨です。
Macの方
  1. ChromeまたはEdgeでEasyKnobを開きます。SafariではOutput選択が使えない場合があります。
  2. BlackHoleの公式配布元から2ch版をインストールします。
  3. インストール後、必要に応じてMacを再起動します。
  4. Audio MIDI設定を開き、BlackHoleが表示されていることを確認します。
  5. EasyKnobのOutputを BlackHole にします。
  6. Discord、VRChat、OBSなどの入力デバイスを BlackHole にします。
  7. EasyKnobのONを押します。初回はここでブラウザのマイク許可が出るので、許可してください。
  8. 外部アプリ側のノイズ抑制、エコー除去、自動ゲイン調整はOFF推奨です。

触らずに音を作る

よく使う音はボタンだけで反映できます。音を出した後に、必要なところだけノブで微調整してください。

画面の見方

基本操作

Input、Output、Presetを選んでONを押します。LIVEと表示されたら動作中です。RESETはプリセットを変えずに、現在のパラメーターだけ戻します。

プリセット

Default、Sing、Talk、Preset 1から3を選べます。変更したノブ値、ON/OFF、REVERB詳細はブラウザに自動保存されます。

REVERB 詳細

REVERBの詳細ボタンから、Hzグラフ上のポイントを追加、削除、ドラッグできます。選択中のポイントはFREQ、GAIN、Qで細かく調整できます。

ノブのON/OFF

各ノブ右上のON/OFF表示を押すと、そのノブだけ一時的に無効化できます。設定画面で無効化したノブは非表示になり、処理からも外れます。

MIC入力音量。0でEasyKnobから出る声も0。 ECHO後ろで小さく返る遅れ音。 REVERB声の後ろに響きを足す。 ROOM空間の広さと左右の広がり。 WETエフェクト音の混ざり具合。 TONE声の明るさ。 AIR高域の抜けと空気感。 STABLE音量差や刺さる部分を抑える。 DOUBLE声を少し重ねて厚みを足す。

さらに詳しいKnob(ノブ)の説明は、設定画面の各項目にも記載しています。

困った時

音が出ない

初回はONを押した後にマイク許可が出ます。ブロックした場合は、ブラウザのサイト設定からEasyKnobのマイクを許可してください。

WindowsはEasyKnobのOutputをCABLE Input、外部アプリの入力をCABLE Outputにします。MacはEasyKnob側も外部アプリ側もBlackHoleが基本です。

遅延・音質

DiscordやOBS側のノイズ抑制、エコー除去、自動ゲイン調整はOFF推奨です。Windowsの音声拡張、空間オーディオ、メーカー製マイク補正もOFFにしてください。

Windowsの「このデバイスを聴く」、Discordのマイクテスト、OBSの音声モニタリングなど、別経路の監視音もOFF推奨です。

ハウリング

スピーカーから出した音をマイクが拾うとハウリングします。基本はヘッドホンか仮想オーディオ出力を使ってください。

HOWLが出る時は、MIC、WET、ECHO、REVERB、DOUBLEを下げ、Outputをスピーカー以外にしてください。

重い・不安定

不要なSTABLE、ECHO、REVERB、ROOM、WET、AIR、DOUBLEをOFFまたは低めにしてください。AUDIO ANALYZERも不要な時はOFF推奨です。

対応外環境について

Safariや一部ブラウザでは、出力先デバイスの選択が使えない場合があります。対応外環境では、EasyKnobを仮想マイク用途に使えない可能性があります。
